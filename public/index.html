Classes Tracker

This is a website for you to input the classes you have taken. You can view what classes you need to take, what classes you have left to take.

This website focuses solely on the reqiurements for a Computer Science major.

The classes that you input are saved to a database. You can then view what you have entered.

Input the class descriptor, such as CSCI or MATH etc, the class number, and professor name.
